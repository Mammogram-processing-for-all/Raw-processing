    LUT (Look-Up Table)¶

    мҡ”м•Ҫ

    LUT(Look-Up Table)лҠ” н”Ҫм…Җлі„лЎң л№„мӢј н•ЁмҲҳ(sigmoid, gamma, log, windowing, GSDF)лҘј л§Ө н”Ҫм…Җ мһ¬кі„мӮ°н•ҳлҠ” лҢҖмӢ , мһ…л Ҙк°’ вҶ’ м¶ңл Ҙк°’ л§Өн•‘мқ„ лҜёлҰ¬ н•ң лІҲ кі„мӮ°н•ҙ л°°м—ҙлЎң м ҖмһҘн•ҙ л‘җкі  н”Ҫм…ҖлӢ№ O(1) мЎ°нҡҢлЎң м Ғмҡ©н•ҳлҠ” мһҗлЈҢкө¬мЎ°лӢӨ. мӣҗлһҳ мқҳлҸ„(л°қкё° мЎ°м •В·ліҙм • мӢң sigmoid к°ҷмқҖ м—°мӮ°мқ„ лҜёлҰ¬ кі„мӮ°н•ҙ мҶҚлҸ„лҘј лҶ’мқҙлҠ” н‘ң)лҘј мң м§Җн•ҳл©ҙм„ң, 1D/ліҙк°„/3D LUTмқҳ мў…лҘҳ, NumPyВ·OpenCV кө¬нҳ„, к·ёлҰ¬кі  DICOMмқҳ Modality / VOI / Presentation LUT нҢҢмқҙн”„лқјмқёк№Ңм§Җ мЎ°л§қн•ңлӢӨ.

    1. LUTлһҖ? вҖ” м •мқҳмҷҖ лҸҷкё°¶

    X-ray мҳҒмғҒ мІҳлҰ¬м—җм„ң л°қкё° мЎ°м •В·ліҙм •мқ„ н•  л•Ң sigmoid function к°ҷмқҖ л№„м„ нҳ• ліҖнҷҳмқ„ мһҗмЈј м“ҙлӢӨ. мқҙлҹ° н•ЁмҲҳлҘј мҲҳл°ұл§Ң к°ң н”Ҫм…Җл§ҲлӢӨ м§Ғм ‘ кі„мӮ°н•ҳл©ҙ л№„мҡ©мқҙ нҒ¬лӢӨ. LUTлҠ” к·ё ліҖнҷҳмқ„ к°ҖлҠҘн•ң лӘЁл“  мһ…л Ҙк°’м—җ лҢҖн•ҙ лҜёлҰ¬ н•ң лІҲ кі„мӮ°н•ҙ л°°м—ҙм—җ лӢҙм•„ л‘җкі , мӢӨм ң мІҳлҰ¬м—җм„ңлҠ” н”Ҫм…Җк°’мқ„ мқёлҚұмҠӨлЎң л°°м—ҙмқ„ мЎ°нҡҢ(lookup)н•ҳкё°л§Ң н•ңлӢӨ.

    \[ y = f(x) \quad\Longrightarrow\quad \text{LUT}[x] = f(x),\quad y = \text{LUT}[x] \]

    мҷң л№ лҘёк°Җ

    16-bit мҳҒмғҒмқҳ к°ҖлҠҘн•ң мһ…л Ҙк°’мқҖ \(2^{16} = 65536\) к°Җм§ҖлҝҗмқҙлӢӨ. мҳҒмғҒмқҙ мҲҳл°ұл§Ң~мҲҳмІңл§Ң н”Ҫм…Җмқҙм–ҙлҸ„ н•ЁмҲҳ нҸүк°ҖлҠ” 65536лІҲл§Ң н•ҳл©ҙ лҗҳкі , мқҙнӣ„лҠ” н”Ҫм…ҖлӢ№ л©”лӘЁлҰ¬ мЎ°нҡҢ 1нҡҢ(O(1))лӢӨ. мҙҲмӣ”н•ЁмҲҳ(exp, pow, log)лҘј н”Ҫм…Җл§ҲлӢӨ л¶ҖлҘҙлҠ” кІғкіј л№„көҗн•ҳл©ҙ мҲҳмӢӯ~мҲҳл°ұ л°° л№ лҘҙлӢӨ. лҚ”кө¬лӮҳ windowingВ·gammaВ·нҠ№м„ұкіЎм„ В·GSDFлҘј н•ҳлӮҳмқҳ н•©м„ұ LUTлЎң лҜёлҰ¬ н•©міҗ л‘ҳ мҲҳ мһҲм–ҙ, м—¬лҹ¬ ліҖнҷҳмқ„ лӢЁмқј мЎ°нҡҢлЎң лҒқлӮј мҲҳ мһҲлӢӨ.

    2. 1D LUT вҖ” к·ёл ҲмқҙмҠӨмјҖмқј нҶӨ л§Өн•‘¶

    к°ҖмһҘ нқ”н•ң нҳ•нғңлӢӨ. мһ…л Ҙ к·ёл Ҳмқҙк°’ н•ҳлӮҳлҘј м¶ңл Ҙ к·ёл Ҳмқҙк°’ н•ҳлӮҳлЎң лҢҖмқ‘мӢңнӮӨлҠ” 1м°Ёмӣҗ л°°м—ҙмқҙлӢӨ. windowing, gamma, к·ёлҰ¬кі  лҢҖл¶Җл¶„мқҳ нҠ№м„ұ кіЎм„ мқҙ 1D LUTлЎң кө¬нҳ„лҗңлӢӨ.

    • нҒ¬кё°: мһ…л Ҙ л№„нҠё к№Ҡмқҙм—җ лҢҖн•ҙ \(2^{\text{bitdepth}}\) м—”нҠёлҰ¬. 8-bitл©ҙ 256, 16-bitл©ҙ 65536.
    • л©”лӘЁлҰ¬: 16-bit мһ…л Ҙ вҶ’ 8-bit м¶ңл Ҙ LUTлҠ” \(65536 \times 1\,\text{byte} = 64\,\text{KB}\). 16-bit мһ…л Ҙ вҶ’ 16-bit м¶ңл Ҙмқҙл©ҙ \(65536 \times 2 = 128\,\text{KB}\). лӘЁл‘җ CPU мәҗмӢңм—җ л“Өм–ҙк°Ҳ л§ҢнҒј мһ‘лӢӨ.

    3. кө¬нҳ„: precompute then index¶

    н•өмӢ¬ нҢЁн„ҙмқҖ "н•ң лІҲ кі„мӮ°(precompute) вҶ’ мқёлҚұмӢұ"мқҙлӢӨ.

    import numpy as np

# 16-bit мһ…л Ҙ -> 8-bit м¶ңл Ҙ gamma LUTлҘј лҜёлҰ¬ кі„мӮ°
gamma = 2.2
lut = (255 * ((np.arange(65536) / 65535) ** gamma)).astype(np.uint8)

# м Ғмҡ©: н”Ҫм…Җк°’мқ„ мқёлҚұмҠӨлЎң мЎ°нҡҢ (fancy indexing = лІЎн„°нҷ”лҗң O(N) мЎ°нҡҢ)
out = lut[img]          # img: uint16 л°°м—ҙ -> out: uint8 л°°м—ҙ

    import numpy as np

def build_window_lut(ww, wc, out_max=255, in_bits=16):
    x = np.arange(1 << in_bits, dtype=np.float64)
    y = (x - (wc - ww / 2)) / ww          # м„ нҳ• л§Өн•‘
    y = np.clip(y, 0, 1) * out_max
    return y.astype(np.uint8)

lut = build_window_lut(ww=44000, wc=30000)
out = lut[img]

    import cv2, numpy as np

# cv2.LUTлҠ” 8-bit мһ…л Ҙл§Ң м§Җмӣҗ (LUT нҒ¬кё° 256 кі м •)
lut8 = (255 * ((np.arange(256) / 255) ** 0.5)).astype(np.uint8)
out8 = cv2.LUT(img8, lut8)               # img8: uint8

# 16-bitлҠ” cv2.LUT л¶Ҳк°Җ -> NumPy м§Ғм ‘ мқёлҚұмӢұ мӮ¬мҡ©
out16 = lut16[img16]

    OpenCV cv2.LUTмқҳ 8-bit м ңн•ң

    cv2.LUTлҠ” LUT кёёмқҙк°Җ 256мңјлЎң кі м •лҗҳм–ҙ 8-bit мһ…л Ҙ м „мҡ©мқҙлӢӨ. мң л°©мҙ¬мҳҒмқҳ 12вҖ“16-bit мҳҒмғҒм—җлҠ” м“ё мҲҳ м—ҶмңјлҜҖлЎң, NumPyмқҳ fancy indexing(lut[img])мңјлЎң м§Ғм ‘ мқёлҚұмӢұн•ҙм•ј н•ңлӢӨ. мқёлҚұмӢұ м „м—җ мһ…л Ҙмқҙ LUT лІ”мң„лҘј лІ—м–ҙлӮҳм§Җ м•ҠлҸ„лЎқ(мҳҲ: мқҢмҲҳ, 65536 мқҙмғҒ) нҒҙлҰ¬н•‘н•ҳлҠ” кІғмқҙ м•Ҳм „н•ҳлӢӨ.

    4. ліҙк°„ LUTмҷҖ 3D LUT¶

    4.1 ліҙк°„ LUT (interpolated LUT)¶

    м „мІҙ мһ…л Ҙ лҸ„л©”мқё лҢҖмӢ  мғҳн”Ң м җ лӘҮ к°ңл§Ң м ҖмһҘн•ҳкі , к·ё мӮ¬мқҙлҠ” м„ нҳ• ліҙк°„(linear interpolation)мңјлЎң мұ„мҡҙлӢӨ. кіЎм„ мқҙ л§ӨлҒ„лҹ¬мҡё л•Ң л©”лӘЁлҰ¬лҘј нҒ¬кІҢ мӨ„мқёлӢӨ. мҳҲлҘј л“Өм–ҙ 65536 м—”нҠёлҰ¬ лҢҖмӢ  256к°ң л…ёл“ңл§Ң м ҖмһҘн•ҳкі  np.interpлЎң 16-bitлҘј ліөмӣҗн•  мҲҳ мһҲлӢӨ.

    nodes_x = np.linspace(0, 65535, 256)
nodes_y = tone_curve(nodes_x)               # л№„мӢј н•ЁмҲҳ 256лІҲл§Ң нҸүк°Җ
out = np.interp(img, nodes_x, nodes_y)      # лӮҳлЁём§ҖлҠ” м„ нҳ• ліҙк°„

    4.2 3D LUT (color)¶

    м»¬лҹ¬ мҳҒмғҒм—җм„ң (R,G,B) вҶ’ (R',G',B') ліҖнҷҳмқ„ мң„н•ң 3м°Ёмӣҗ кІ©мһҗ LUTлӢӨ. мҳҒнҷ”В·мӮ¬м§„ мғүліҙм •м—җ л„җлҰ¬ м“°мқҙл©° кІ©мһҗм җ мӮ¬мқҙлҘј мӮјм„ нҳ•(trilinear) ліҙк°„н•ңлӢӨ. мң л°©мҙ¬мҳҒмқҖ ліём§Ҳм ҒмңјлЎң к·ёл ҲмқҙмҠӨмјҖмқјмқҙлқј 3D LUTлҠ” л“ңл¬јлӢӨ вҖ” мқҳмӮ¬мғү(pseudocolor) н‘ңмӢңмІҳлҹј нҠ№мҲҳн•ң кІҪмҡ°м—җл§Ң л“ұмһҘн•ңлӢӨ.

    5. DICOMмқҳ LUT нҢҢмқҙн”„лқјмқё¶

    DICOM к·ёл ҲмқҙмҠӨмјҖмқј н‘ңмӢңлҠ” м„ё лӢЁкі„мқҳ LUTлҘј мҲңм„ңлҢҖлЎң м Ғмҡ©н•ңлӢӨ. к°Ғ лӢЁкі„лҠ” м•һ м Ҳм—җм„ң ліё 1D LUTлЎң кө¬нҳ„лҗңлӢӨ.

    VOI LUT Value Of Interest м„ нғқ = windowing. Window Width/Center (0028,1051/1050) м Ғмҡ©. windowing
    Presentation LUT н‘ңмӢң мһҘм№ҳм—җм„ң мқём§Җм ҒмңјлЎң к· мқјн•ҳлҸ„лЎқ GSDF м Ғмҡ©. л””мҠӨн”Ңл Ҳмқҙ GSDF

    GSDFк°Җ Presentation LUTлЎң м Ғмҡ©лҗңлӢӨ

    л””мҠӨн”Ңл Ҳмқҙ GSDFм—җм„ң м„ӨлӘ…н•ң Grayscale Standard Display FunctionмқҖ Presentation ValueлҘј лӘЁлӢҲн„° нңҳлҸ„лЎң л§Өн•‘н•ҳлҠ” н‘ңмӨҖ кіЎм„ мқҙлӢӨ. мқҙ кіЎм„ мқҙ л°”лЎң presentation LUTлЎң кө¬нҳ„лҗҳм–ҙ, нңҳлҸ„ нҠ№м„ұмқҙ лӢӨлҘё лӘЁлӢҲн„°л“Өм—җм„ң к°ҷмқҖ мҳҒмғҒмқҙ мқјкҙҖлҗҳкІҢ ліҙмқҙлҸ„лЎқ н•ңлӢӨ. мҰү GSDFлҠ” "лҠҗлҰ° мқём§Җ кіЎм„ мқ„ лҜёлҰ¬ кі„мӮ°н•ҙ л‘җлҠ”" м „нҳ•м Ғмқё LUT мқ‘мҡ©мқҙлӢӨ.

    6. н•ңкі„мҷҖ мЈјмқҳ¶

    1D LUTлҠ” м „м—ӯ(global) м җ м—°мӮ°мқҙлӢӨ

    LUTлҠ” мһ…л Ҙк°’м—җл§Ң мқҳмЎҙн•ҳкі  мң„м№ҳ(кіөк°„ мўҢн‘ң)лҠ” ліҙм§Җ м•ҠлҠ”лӢӨ. л”°лқјм„ң к°ҷмқҖ мһ…л Ҙк°’мқҖ мҳҒмғҒ м–ҙл””м—җ мһҲл“  к°ҷмқҖ м¶ңл ҘмңјлЎң л§Өн•‘лҗңлӢӨ. л‘җк»ҳ кө¬л°°лӮҳ peripheral thinningмІҳлҹј кіөк°„м ҒмңјлЎң ліҖн•ҳлҠ” ліҙм •мқҖ 1D LUTлЎң н‘ңнҳ„н•  мҲҳ м—ҶлӢӨ.

    мқҙлҹ° кіөк°„ к°ҖліҖ мІҳлҰ¬лҠ” LUTк°Җ м•„лӢҲлқј лӢӨмқҢ кё°лІ•мқҙ лӢҙлӢ№н•ңлӢӨ.

    • CLAHE(Contrast Limited Adaptive Histogram Equalization): нғҖмқјлі„лЎң лӢӨлҘё л§Өн•‘мқ„ м Ғмҡ© (м җ м—°мӮ°мқҳ нҶӨ кіЎм„ мқ„ көӯмҶҢнҷ”н•ң кІғ).
    • көӯмҶҢ/м Ғмқ‘ мІҳлҰ¬: peripheral equalization, лӢӨмӨ‘мҠӨмјҖмқј л¶„н•ҙ кё°л°ҳ л””н…Ңмқј к°•мЎ° л“ұ.

    л”°лқјм„ң мӢӨм ң нҢҢмқҙн”„лқјмқёмқҖ м „м—ӯ ліҖнҷҳмқҖ LUTлЎң к°ҖмҶҚн•ҳкі , кіөк°„ к°ҖліҖ ліҙм •мқҖ лі„лҸ„ көӯмҶҢ мІҳлҰ¬лЎң лӮҳлҲ„м–ҙ мҲҳн–үн•ңлӢӨ.

    7. м •лҰ¬¶

    LUT мў…лҘҳ мһ…л ҘвҶ’м¶ңл Ҙ нҒ¬кё°(16-bit кё°мӨҖ) мҡ©лҸ„
    1D мҷ„м „ 1ch вҶ’ 1ch 65536 м—”нҠёлҰ¬ gamma, windowing, нҶӨ кіЎм„ , GSDF
    1D ліҙк°„ 1ch вҶ’ 1ch мҲҳмӢӯ~мҲҳл°ұ л…ёл“ң + ліҙк°„ л§ӨлҒ„лҹ¬мҡҙ кіЎм„ мқҳ л©”лӘЁлҰ¬ м Ҳм•Ҫ
    3D 3ch вҶ’ 3ch \(N^3\) кІ©мһҗ + мӮјм„ нҳ• ліҙк°„ м»¬лҹ¬ ліҖнҷҳ(л§ҳлӘЁм—җм„  л“ңл¬ҫ)

    LUTмқҳ мһ…л Ҙ кіЎм„ л“ӨмқҖ нҠ№м„ұ кіЎм„ м—җм„ң м •мқҳлҗҳкі , лҢҖн‘ң мқ‘мҡ©мқё windowingмқҖ windowingм—җм„ң лӢӨлЈ¬лӢӨ.
